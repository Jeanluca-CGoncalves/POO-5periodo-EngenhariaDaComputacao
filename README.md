# 💻 Programação Orientada a Objetos - 5º Período
### Engenharia da Computação |

Repositório destinado ao armazenamento de projetos e exercícios desenvolvidos durante a disciplina de POO. O foco principal é a aplicação de conceitos fundamentais e avançados de Engenharia de Software utilizando a linguagem Java.

---

## 🚀 Tecnologias e Conceitos Aplicados
* **Linguagem:** Java ☕
* **Paradigma:** Orientado a Objetos (Encapsulamento, Herança, Polimorfismo e Abstração).
* **Concorrência:** Gerenciamento de Threads (Simulações Bancárias).
* **Estrutura de Dados:** Manipulação de Strings e Objetos Complexos.

## 📂 Projetos no Repositório

| Projeto | Descrição | Conceito Chave |
| :--- | :--- | :--- |
| **BancoThreads** | Simulação de depósitos e saques simultâneos. | Multithreading / Sincronização |
| **RH / InfoEmpregado** | Sistema de gestão de funcionários e cargos. | Polimorfismo e Herança |
| **Concessionaria** | Modelagem de veículos (Carro, Caminhão, Moto). | Abstração e Classes Base |
| **AreaPeri / ContaBancaria** | Cálculos geométricos e lógica financeira básica. | Encapsulamento |
| **Conversão / Reversão** | Utilitários para temperatura e manipulação de strings. | Lógica de Programação |
